# Infraestructura Windows Server

En este repositorio recopilo algunas de las prácticas de administración de sistemas Windows que he realizado durante mi formación en Administración de Sistemas Informáticos en Red (ASIR).

He seleccionado aquellas actividades que considero más representativas de los conocimientos adquiridos en Active Directory, administración de usuarios y grupos, almacenamiento, permisos y directivas de grupo.

## Contenido

### [01 - Infraestructura Active Directory](./01%20-%20Infraestructura%20Active%20Directory.pdf)

Práctica centrada en la instalación y configuración de Active Directory, creación de usuarios, grupos y unidades organizativas (OU), así como la administración básica de un entorno de dominio.

### [02 - Almacenamiento, cuotas y RAID en Windows](./02%20-%20Almacenamiento,%20cuotas%20y%20RAID%20en%20Windows.pdf)

Práctica orientada a la gestión de almacenamiento mediante particiones, sistemas de archivos NTFS, cuotas de disco y configuraciones RAID.

### [03 - Caso práctico de administración de Windows Server](./03%20-%20Caso%20práctico%20de%20administración%20de%20Windows%20Server.pdf)

Resolución de un escenario empresarial completo que incluye Active Directory, organización de usuarios y grupos, recursos compartidos, permisos NTFS y aplicación de directivas de grupo (GPO).

## Tecnologías y conceptos trabajados

* Windows Server
* Active Directory
* Unidades Organizativas (OU)
* Usuarios y grupos
* AGDLP
* NTFS
* Recursos compartidos
* Directivas de Grupo (GPO)
* GPT
* Cuotas de disco
* RAID
* VirtualBox

## Sobre este repositorio

Mi objetivo es utilizar este espacio para mostrar algunas de las prácticas que mejor representan mi aprendizaje en administración de sistemas Windows durante el primer curso de ASIR.
